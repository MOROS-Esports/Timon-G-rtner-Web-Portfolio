Über mich

Ich programmiere in meiner Freizeit und versuche immer Neues zu lernen. Dazu haben bisher die Programmiersprachen Java, C# und Python gehört. Während Ich diese Website erstelle übe ich mich an Web Developement und damit an HTML5, CSS und PHP.

In diesem Web-Portfolio erfährst du mehr über mich und meine bisherigen Projekte.
